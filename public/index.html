Tradutor para Libras

  1. Clique em Choose File
  2. Na nova caixa de diálogo, selecione o arquivo de texto (.txt) a ser mostrado
  3. Clique em Enviar
  4. Abra o Widget do VLibras clicando no botão azul do lado direito
  5. Clique no ícone de engrenagem no canto superior esquerdo do Widget
  6. Ajuste o slider de Opacidade para 0
  7. Ainda vou pesquisar a personalização do avatar. É apenas um protótipo!!
